Intermediate Level

Day 1-3: Advanced Network Scanning and Enumeration

  • Total time: 6 hours/day

  • Hour 1 - 2: Nmap Advanced Scanning
    • Learn advanced Nmap techniques like evading firewalls, IDS/IPS, and conducting stealth scans.
    • Use options like nmap -sS, -sF, and --data-length to bypass filters.
    • Conduct scans with -Pn, -T, and --reason to gather more detailed information while avoiding detection.

  • Hour 3: OS Detection and Version Scanning
    • Understand fingerprinting techniques to detect operating systems and software versions.
    • Practice using Nmap and Xprobe2 for advanced OS detection: nmap -O [target_IP].

  • Hour 4 - 5: Advanced Enumeration with Enum4Linux and SNMP
    • Use Enum4Linux for SMB enumeration: gather information about users, shares, and services.
    • Practice using SNMPWalk to enumerate network devices using SNMP.

  • Hour 6: Vulnerability Scanning
    • Use tools like OpenVAS or Nessus to scan for vulnerabilities on your target systems.
    • Understand how to read scan results and prioritize vulnerabilities.


Day 4-6: Web Application Penetration Testing (Intermediate)

  • Total time: 6 hours/day

  • Hour 1 - 2: Manual Testing for Web Vulnerabilities
    • Perform manual testing for SQL Injection, XSS, and CSRF vulnerabilities without automated tools.
    • Use browser tools (like Burp Suite) and manual input fields to identify vulnerabilities.

  • Hour 3 - 4: Advanced SQL Injection
    • Move beyond basic SQLi and practice blind SQL injection, time-based injections, and extracting data using UNION queries.
    • Use tools like SQLMap for automation and analyze how it works internally.

  • Hour 5: Authentication and Session Management
    • Learn about authentication flaws and test for session management vulnerabilities.
    • Explore cookie manipulation, session fixation, and insecure password storage.

  • Hour 6: Bypassing Web Application Firewalls (WAF)
    • Understand how WAFs work and learn techniques for bypassing them (e.g., encoding, using special characters, etc.).
    • Use tools like WAFW00F to detect the presence of a WAF.


Day 7-9: Exploit Development (Intermediate)

  • Total time: 6 hours/day

  • Hour 1 - 2: Buffer Overflow Basics
    • Dive into buffer overflows and learn how they work in-depth.
    • Write basic buffer overflow exploits using tools like Immunity Debugger and gdb.

  • Hour 3: SEH (Structured Exception Handler) Overwrite
    • Learn how to exploit SEH-based buffer overflows by overwriting exception handlers in Windows.
    • Practice exploiting vulnerable applications that use SEH.

  • Hour 4 - 5: Exploit Research and CVE
    • Research public exploits (using Exploit-DB or CVE databases) and understand how they work.
    • Download and test exploits on vulnerable software.

  • Hour 6: Shellcoding
    • Learn shellcoding techniques to craft custom shellcode payloads.
    • Use tools like msfvenom to generate payloads and embed them into exploits.


Day 10-12: Wireless and IoT Hacking

  • Total time: 6 hours/day

  • Hour 1 - 2: Wireless Hacking Basics
    • Understand wireless security protocols (WEP, WPA, WPA2).
    • Use Aircrack-ng to capture wireless traffic and attempt to crack WEP and WPA keys.

  • Hour 3: Evil Twin Attack
    • Set up an evil twin attack to trick users into connecting to your rogue access point.
    • Use tools like airbase-ng and Wireshark to capture credentials.

  • Hour 4 - 5: Bluetooth Hacking
    • Learn how to exploit vulnerabilities in Bluetooth using tools like BlueHydra or Bluesnarfer.
    • Practice discovering and attacking Bluetooth-enabled devices.

  • Hour 6: IoT Device Hacking
    • Explore security vulnerabilities in IoT devices.
    • Use Binwalk and Firmadyne to analyze IoT firmware for backdoors or other flaws.


Day 13-15: Privilege Escalation and Post-Exploitation

  • Total time: 6 hours/day

  • Hour 1 - 2: Linux Privilege Escalation
    • Learn Linux privilege escalation techniques: kernel exploits, misconfigured SUID files, cron jobs, etc.
    • Practice using tools like LinPEAS to find weaknesses.

  • Hour 3: Windows Privilege Escalation
    • Understand Windows privilege escalation: insecure services, DLL hijacking, unquoted paths, and more.
    • Use PowerUp and WinPEAS to automate discovery of privilege escalation paths.

  • Hour 4 - 5: Post-Exploitation with Metasploit
    • Perform post-exploitation tasks like gathering credentials, escalating privileges, and maintaining access on compromised systems.
    • Practice persistence methods (e.g., backdoors, rootkits).

  • Hour 6: Lateral Movement
    • Learn how to move laterally across networks by exploiting trusts between machines (e.g., pass-the-hash or RDP connections).
    • Use PsExec to execute commands on remote systems after credential compromise.


Day 16-18: Advanced Scripting and Automation

  • Total time: 6 hours/day

  • Hour 1 - 2: Python for Automation
    • Write Python scripts to automate scanning, enumeration, or exploitation tasks.
    • Practice using Python libraries like Scapy, Socket, and Requests.

  • Hour 3: Bash Scripting for Penetration Testing
    • Write Bash scripts to automate repetitive tasks like network scanning, data parsing, and reporting.

  • Hour 4 - 5: PowerShell for Post-Exploitation
    • Use PowerShell to automate post-exploitation on Windows machines.
    • Practice running PowerShell scripts for tasks like downloading files, gathering credentials, and escalating privileges.

  • Hour 6: Cron Jobs and Scheduled Tasks
    • Learn how to set up persistent cron jobs on Linux and scheduled tasks on Windows to automate and maintain access.


Day 19-21: Capture the Flag (CTF) and Challenges

  • Total time: 6 hours/day

  • Hour 1 - 2: CTF Preparation
    • Participate in intermediate-level CTFs on platforms like TryHackMe or Hack The Box.
    • Focus on a range of challenges: web exploitation, network attacks, reverse engineering, etc.

  • Hour 3 - 4: Web Exploitation Challenges
    • Solve web-related CTF challenges that require knowledge of advanced vulnerabilities like SQLi, XSS, and CSRF.

  • Hour 5 - 6: System Exploitation Challenges
    • Focus on privilege escalation, lateral movement, and post-exploitation challenges in CTFs.


Weekly Goals:

  • Master network scanning and enumeration techniques beyond the basics.
  • Gain deeper understanding of web app security, including manual testing and advanced attacks.
  • Learn and practice exploit development, buffer overflows, and shellcoding.
  • Understand wireless hacking and exploit IoT devices.
  • Practice privilege escalation, post-exploitation, and automation using scripts.
  • Participate in intermediate-level CTFs to reinforce skills.

Beginning Level

Day 1-3: Setting Up Your Environment and Basic Linux Skills

  • Total time: 6 hours/day

  • Hour 1 - 2: Set Up Virtual Machines (VMs)
    • Install Kali Linux and a vulnerable machine like Metasploitable2 using VMware or VirtualBox.
    • Familiarize yourself with VM settings and network configurations (NAT, Bridged, Host-only).

  • Hour 3: Introduction to Kali Linux
    • Basic commands in Kali: ls, cd, pwd, cp, mv, rm, chmod, chown, apt update, apt install.
    • Practice navigating the Linux filesystem and understand basic file permissions.

  • Hour 4 - 5: Networking Basics
    • Learn about IP addresses, DNS, TCP/UDP, and ports. Use ifconfig and ip a to check your IP.
    • Use tools like ping, traceroute, and netstat to explore basic network behavior.

  • Hour 6: Basic Network Scanning with Nmap
    • Introduction to Nmap. Perform basic scans to discover open ports on the Metasploitable2 machine: 
       nmap -sP [target_IP]
       nmap -sV [target_IP].


Day 4-6: Reconnaissance and Enumeration

  • Total time: 6 hours/day

  • Hour 1 - 2: Information Gathering
    • Use whois, dig, and nslookup to gather information about domains and IP addresses.
    • Research OSINT (Open-Source Intelligence): Use websites like Shodan and Google Dorking to find vulnerable systems.

  • Hour 3 - 4: Advanced Scanning with Nmap
    • Learn advanced Nmap options like -A, -O for OS detection, and -sC to run default scripts.
    • Perform a detailed scan of your Metasploitable2 VM: nmap -A [target_IP]

  • Hour 5: Banner Grabbing
    • Learn banner grabbing techniques using Netcat and Telnet to extract information from open ports.
    • Example: 
      nc [target_IP] 80 for HTTP headers

  • Hour 6: Service Enumeration
    • Identify services running on open ports using tools like Nikto for web servers and Enum4Linux for SMB shares.


Day 7-9: Vulnerability Identification and Exploitation Basics

  • Total time: 6 hours/day

  • Hour 1 - 2: Common Vulnerabilities
    • Learn about common vulnerabilities: SQL Injection, Cross-Site Scripting (XSS), Buffer Overflow.
    • Understand the OWASP Top 10 security risks.

  • Hour 3: SQL Injection (Basic)
    • Set up an intentionally vulnerable web app like DVWA.
    • Practice SQL Injection: bypass login using ' OR 1=1--

  • Hour 4 - 5: Using Metasploit Framework
    • Learn how to launch Metasploit (msfconsole).
    • Perform a basic exploit: Find a vulnerability in Metasploitable2, use Metasploit to launch the exploit, and gain a shell.

  • Hour 6: Privilege Escalation
    • Understand privilege escalation basics. Use LinEnum or linux-exploit-suggester on your compromised system to find potential ways to escalate privileges.

  • Hour 6: Clean Up
    • Learn how to clean up after a penetration test. Remove backdoors, tools, and logs to restore the system to its previous state.

Weekly Goals:

  • Understand and set up your hacking environment.
  • Learn basic Linux commands, networking, and web vulnerabilities.
